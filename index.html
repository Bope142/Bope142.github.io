1. Présentation de l'application BCHAT

BCHAT est une application de messagerie Android utilisant le Bluetooth de votre téléphone afin d’échanger des messages avec d’autres utilisateurs de l’application BCHAT.

BCHAT en abrégé, signifie Bluetooth Chat, un projet open-source où j’ai mis à votre disposition les sources de cette dernière.

L’application a été réalisé afin de servir d’exemple d’utilisation non seulement pour toutes les personnes désirantes se lancer dans la programmation réseau avec le langage pascal via l’EDI Delphi tout en utilisant le moyen de communication Bluetooth mais aussi pour les différentes personnes désirantes communiquer entre eux sans toutefois faire usage des forfaits internet.
Ce projet open source exploite le grand concept de la programmation, entre autres la programmation réseau ,la gestion des bases des données dans une application Android d’une manière locale.

2. Comment utiliser BCHAT pour communiquer gratuitement avec vos proches

BBCHAT étant une application de messagerie gratuite vous permettant d’échanger de message avec vos proches sans toutefois avoir une connexion internet, simplement avec le Bluetooth de votre téléphone.

L’application est facile à utiliser :

1. Télécharger le fichier Apk ici, et installer sur votre téléphone ;

2. Une fois l’application installée sur votre téléphone, lancer l’application ;

3. Insérez votre nom dans le champ nom d’utilisateur ;

4. Insérez aussi votre adresse Bluetooth dans le champ adresse Bluetooth (cette adresse est complètement privée, elle ne sera pas partagée avec d’autres utilisateur, c’est juste un simple identifiant dans votre réseau chat. Obtenez votre adresse Bluetooth en allant dans l’option paramètre de votre téléphone et dans le menu à propos.) ;

5. Cliquez sur choisir un profil, afin de choisir un profil avatar (Notons que dans cette première version du projet Open-source je n’ai pas encore mis la possibilité de choisir sa photo se trouvant sur votre téléphone comme profil, dans les versions je pourrais ajouter cette option, notons que le projet étant open-source vous pouvez aussi personnaliser l’application par rapport à vos besoins vu que c’est une application que j’ai codé dans peu de temps avec un timing vraiment réduit) …

6. Une fois ces informations renseignées, vous pouvez alors cliquer sur ENREGISTRER pour sauvegarder ces informations de manière permanente.

3. Lancer une discussion avec un autre utilisateur de BCHAT

Une fois les précédentes étapes remplies vous pouvez discuter avec d’autres utilisateurs de l’application BCHAT.

Pour discuter avec d’autres utilisateur de BCHAT, il faut :

1. Vous rassurez que votre Bluetooth soit actuellement activé ;

2. Vous rassurez que le téléphone dont vous désirez discuter avec doit être aussi appairé parmi vos différents périphériques Bluetooth disponible dans la liste de votre téléphone (si ce n’est pas le cas, il sera important de le faire avant de pouvoir communiquer avec cet utilisateur) ;

3. Cliquez ensuite sur la bulle de discussion dans l’application afin d’afficher la liste de tous les périphériques déjà jumelés avec votre téléphone ;
